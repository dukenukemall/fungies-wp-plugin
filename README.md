# Fungies WP Plugin

A WordPress plugin for digital selling powered by [Fungies](https://fungies.io).

## Description

This plugin integrates Fungies digital selling capabilities directly into your WordPress site.

## Installation

1. Download the plugin zip file
2. Go to WordPress Admin > Plugins > Add New > Upload Plugin
3. Upload the zip file and activate

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
